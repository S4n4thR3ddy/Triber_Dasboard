Step 1 — Create GitHub Token
1
Open GitHub on your phone browser and log in to github.com/s4n4thr3ddy
2
Go to Settings → Developer Settings → Personal Access Tokens → Fine-grained tokens
3
Click Generate new token. Set name: Triber OBD Logger. Expiry: 1 year.
4
Under Repository access → select Only select repositories → choose Triber_Dasboard
5
Under Permissions → Repository permissions → set Contents to Read and write. That's the ONLY permission needed.
6
Click Generate token → copy it immediately (shown once only!) → paste into Setup tab.
Step 2 — What Gets Saved
Every 15 minutes, a JSON file is created in your repo at:
obd-logs/session_YYYYMMDD_HHMMSS.json
Each file contains:
• Complete sensor snapshot (all PIDs)
• Fault codes active at that moment
• Session stats (max RPM, max catalyst temp)
• Health score + diagnosis summary
• Fuel trim history
• Timestamp + odometer estimate

How to share with Claude for analysis:
Go to Reports tab → tap any report → tap "Copy JSON" → paste to Claude and ask for analysis.
Step 3 — Daily Workflow
🌅 Morning — Connect ELM327, open Logger, tap START before driving

🚗 While driving — Logger auto-saves every 15 min silently

🌙 Evening — Go to Reports tab, tap latest report, review health score

📊 Weekly — Copy last 7 reports, paste to Claude for trend analysis

🔧 If warning appears — Tap "Save Now" to capture the fault immediately
